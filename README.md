# Flow++

A lightweight inline suggestion engine for C and C++ in VS Code and Cursor.

Flow++ provides ghost-text completions as you type. It pairs with clangd, which handles diagnostics, signatures, and semantic completions. Flow++ handles the fast, prefix-driven suggestions for the standard library, preprocessor directives, keywords, and `std::ranges::` algorithms.

---

## Features

- Inline ghost-text completions triggered after 3 characters
- Covers C (C90 through C23) and C++ (C++11 through C++26)
- Full `std::` algorithm coverage
- Full `std::ranges::` algorithm coverage (C++20, C++23, C++26)
- Preprocessor directives and standard headers for both C and C++
- Debounced suggestions to avoid flickering while typing
- Language resolved by file extension — `.h` and `.hpp` handled correctly
- Toggle on/off via the status bar, command palette, or extension settings
- feature coverage.txt shows all what is covered

## Requirements

- VS Code `1.85.0` or later, or Cursor
- Node.js `18.x` or later (for building from source)
- `vsce` (for packaging from source)

---

## Building from Source

Clone the repository and install dependencies in the following order:

```bash
npm install
npm run compile
```

Package the extension:

```bash
vsce package
```

Install into Cursor:

```bash
cursor --install-extension flowplusplus-0.1.0.vsix
```

Or install into VS Code:

```bash
code --install-extension flowplusplus-0.1.0.vsix
```

## Configuration

| Setting | Default | Description |
|---|---|---|
| `flowplusplus.enabled` | `true` | Enable or disable inline suggestions |
| `flowplusplus.triggerLength` | `3` | Minimum characters before suggestions fire |
| `flowplusplus.delay` | `400` | Debounce delay in milliseconds |

## Usage

Start typing any C or C++ construct and ghost text will appear inline. Press `Tab` to accept.

#### Includes
```cpp
#inc               →   #include 
#inq               →   #include "
#pra               →   #pragma once
```

#### Keywords
```cpp
noe                →   noexcept
con                →   const
tem                →   template<typename
```

#### Functions
```cpp
std::ranges::fol   →   std::ranges::fold_left(
std::fin           →   std::find(
mem                →   memcpy(
```

Click the `$(sparkle) Flow++` item in the status bar to toggle suggestions on or off.
